ADN Components On this page

Recorder

Voice recorder with preview, waveform, and single-track upload to AudioDN.

Default

Tap the mic to start a 3‑2‑1 countdown, then it records; tap again to stop, then preview and confirm. Requires an API key + collection ID.

<audiodn-recorder
  api-key="YOUR_API_KEY"
  collection-id="COLLECTION_ID"
  accent-color="#7c3aed"
  theme="light"
  countdown="3"
></audiodn-recorder>

Locale

Same locale attribute as the player and uploader (en, fr, es, de). Listen for file-uploaded when the PUT completes.

<audiodn-recorder
  api-key="YOUR_API_KEY"
  collection-id="COLLECTION_ID"
  locale="fr"
  countdown="2"
></audiodn-recorder>

<script type="module">
  document.querySelector('audiodn-recorder')
    .addEventListener('file-uploaded', (e) => {
      console.log(e.detail.trackId, e.detail.fileName)
    })
</script>

Themes

Override CSS custom properties on the host to restyle the recorder. Uses the same --adn-* API as the uploader.

Midnight
.theme-recorder-midnight {
  --adn-bg: #1a1a2e;
  --adn-bg-light: #2a2a4e;
  --adn-color-font: #e0e0e0;
  --adn-color-font-muted: #8888aa;
  --adn-color-accent: #7c3aed;
  --adn-color-accent-rgb: 124, 58, 237;
  --adn-color-accent-alt: #fff;
  --adn-border-color: #3a3a5e;
  --adn-color-highlight: rgba(124, 58, 237, 0.12);
  --adn-radius: 16px;
  --adn-main-border: 1px solid #3a3a5e;
  --adn-box-shadow: 0 4px 24px rgba(0, 0, 0, 0.4);
}
<audiodn-recorder
  class="theme-recorder-midnight"
  api-key="YOUR_API_KEY"
  collection-id="COLLECTION_ID"
></audiodn-recorder>
